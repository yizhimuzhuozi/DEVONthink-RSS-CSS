# DEVONthink-RSS-CSS
DEVONthink RSS CSS 阅读使用。

### 字体
Roboto，汉仪润圆，Apple Color Emoji

## 使用方法
下载 time.css 文件
复制 time.css 文件到 ~/Library/Application Support/DEVONthink 3/StyleSheets 目录下，然后在「Preference...」-「RSS」的「Feed Style Sheet」中选择「time.css」即可。
